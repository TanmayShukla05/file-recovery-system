File Recovery System

What does this do?

This tool uses a Hopfield Network to memorise the files you upload. Later, if a file is accidentally edited, partially corrupted, overwritten, or otherwise altered, you can upload that modified version and the network will recognise it and recover the closest matching original from its memory.

How to use it:

  1. Store Files: Upload the original files and click "Store". The memory file (hopfield_memory.npz) will download automatically. Save it somewhere safe.
    Important: The server only keeps data temporarily. The downloaded memory file is what lets you retrieve later.
  2. Retrieve Later: If a file becomes corrupted or altered, return here. Upload your saved hopfield_memory.npz and one or more damaged files, then click "Retrieve". You can query several files at once.
  3. Download Result: Each matched file gets its own download link.

Supported File Types

  • Images: PNG, JPG, JPEG, BMP, GIF, WEBP
  • Python Code: PY
  • Jupyter Notebooks: IPYNB
  • Data Files: HDF5, H5, PKL
  • Documents: PDF

How the pattern size is chosen

The pattern size is set automatically so that the load factor α = files / pattern_size stays ≤ {{ alpha_max }}. This keeps the Hopfield Network in a reliable recall regime. Uploading more files grows the pattern size automatically, re-encoding everything at the new resolution.
