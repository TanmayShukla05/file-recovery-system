File Recovery System

What does this do?

This tool uses a Hopfield Network to memorize the files you upload. Later, if a file is accidentally edited, partially corrupted, overwritten, or otherwise altered, you can upload that modified version and the network will try to recognize it and recover the closest matching original file from its memory.

How to use it:

  1. Store Files: Upload the original files you want the network to memorize and click "Store".
  2. Download Memory File: Click "Download Memory File" and save the hopfield_memory.json file to your computer.
    Important: The server only keeps data temporarily and may delete stored files after inactivity. Downloading the memory file ensures you can use the tool later.
  3. Retrieve Later: If a file becomes corrupted, accidentally modified, partially lost, or otherwise differs from the original, return to this page. Upload both your saved hopfield_memory.json file and the altered file, then click "Retrieve".
  4. Download Result: If the network finds a match, you can download the recovered version of the original file.

Supported File Types

  • Images: PNG, JPG, JPEG, BMP, GIF, WEBP
  • Python Code: PY
  • Jupyter Notebooks: IPYNB
  • Data Files: HDF5, H5, PKL
  • Documents: PDF
